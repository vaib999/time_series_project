There are 4 files in the zipped folder.

1) lstm_arima.py(For overall prediction)
2) lstm_arima_indi.py(For individual prediction)
3) result_matrix.txt
4) README.txt

While running lstm_arima_indi.py(2) python file there will be command prompt for input value. You can type value from 0 to 99 for products 1 to 100 and output will be generated.

Requirements for running the code

1) Python 3.5
Modules required:
2)pandas
3)sklearn
4)keras
5)math
6)matplotlib
7)numpy
8)statsmodels
9)tensorflow backend(It only works for Python 3.5 in windows)
